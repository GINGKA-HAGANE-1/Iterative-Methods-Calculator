Numerical Integration

Examples:

• Trigonometric: sin(x), cos(x), tan(x)
• Inverse trig: arcsin(x), arccos(x), arctan(x)
• Constants: Use π or pi for Pi (e.g., sin(π*x))
• Exponential: Use e^x for eˣ
• Logarithmic: ln(x) for natural log, log(x) for log₁₀
• Basic: x^2, x^3, sqrt(x)
• Combined: e^x * sin(x), x^2 * cos(π*x)
